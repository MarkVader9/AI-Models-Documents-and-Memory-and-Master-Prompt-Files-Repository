# 01__JMHZ__OFICIALNI_ZDROJE

Tato část je **pouze pro oficiální dodávky** MPSV/ČSSZ k JMHZ (XSD, PDF, HTML, číselníky, konstanty, metodiky…).

## Zásady
- **Nijak neupravovat obsah dodávky.**  
  Pokud je potřeba anotace, dělej ji v projektové dokumentaci (03__PROJEKTOVE_RIZENI), ne přímo v dodaných souborech.
- **Struktura uvnitř verze respektuje pilíře MPSV/ČSSZ.**
- Každá nová dodávka = nová verze / datované vydání.

## Co sem patří
- ZIPy a jejich rozbalené složky *tak, jak byly dodány*.
- Oficiální dokumentace a popisy.
- „Přehled stromové struktury“ a další meta‑materiály.

## Co sem nepatří
- Naše VB6 soubory ani naše XML výstupy (to patří do 02__SALARY__IMPLEMENTACE).

## Přehled pilířů (kódy)

V této struktuře používáme názvy pilířů tak, aby byly **1:1 dohledatelné** vůči dodávkám MPSV/ČSSZ:

- **REGZEL** — Registrace zaměstnavatele  
- **MH** — Měsíční hlášení  
- **REGZEC** — Registrace zaměstnance  
- **DZMH** — Dotaz na stav zpracování
